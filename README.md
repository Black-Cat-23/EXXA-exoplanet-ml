# EXXA Exoplanet ML

Machine learning experiments for the ML4Sci EXXA project.

## Objectives

- Cluster protoplanetary disk images
- Simulate and classify exoplanet transit curves
- Explore machine learning techniques for astronomical data

## Tech Stack

Python  
NumPy  
Scikit-learn  
Astropy  
Matplotlib  
PyTorch (optional)

## Repository Structure

notebooks/ – experiments and pipelines  
src/ – reusable modules  
plots/ – generated figures  
models/ – trained models
data/- astronmical datsets(.fits)
